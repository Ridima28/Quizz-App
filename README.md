# 🧠 React Quiz App

A simple and interactive quiz application built using **React.js** and **React Router DOM**.  
Users can answer multiple-choice questions, track their score, and view their final result on a dedicated end page.

---

## ✨ Features

- 📝 Multiple-choice quiz questions
- ⏭️ Next and Previous navigation
- ✅ Answer validation with correct/incorrect feedback
- 📊 Real-time score tracking
- 🎯 Final score display on End Page
- 🔄 Restart quiz functionality
- 🚪 End quiz navigation page
- 🌐 React Router based navigation

---

## 🛠️ Tech Stack

- React.js (Functional Components)
- React Hooks (useState, useNavigate, useLocation)
- React Router DOM
- JavaScript (ES6+)
- CSS (Custom styling)

## ⚙️ How It Works

1. User starts at Welcome Page
2. Clicks "Start Quiz"
3. Navigates to Quiz Page
4. Answers questions one by one
5. Clicks "End Quiz"
6. Final score is displayed on End Page
